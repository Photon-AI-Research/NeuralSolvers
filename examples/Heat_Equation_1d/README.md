# PINN example of a 1D Heat Equation


## How to run:

python 1D_Heat_Equation.py --num_epochs=10000 --n0=50 --nb=50 --nf=20000 --num_hidden=4 --hidden_size=100

OR simply the default: python 1D_Heat_Equation.py
